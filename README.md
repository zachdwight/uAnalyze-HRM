# uAnalyze for HRM

uAnalyze is a web-based application for comparison and overlay of experimentally obtained high resolution melting data and theory based melting prediction.  Fun project with D3, Bootstrap and Javascript for the math/IP required for analysis.

## Features

- **Data Visualization**: Plot and overlay multiple melting curves
- **Normalization**: Baseline and exponential normalization options
- **Derivative & Difference Calculations**: Analyze curve characteristics
- **2D Offset Analysis**: Compare curves with reference curve alignment
- **Hierarchical Clustering** *(New)*: Automatically group similar curves using user-specified cluster count (k)
- **Multiple File Format Support**: HR1, LS32, LC480, LC96, LS96/CFX96, Rotorgene, QuantStudio, and Generic formats
- **Predicted Melting Curves**: Add theoretical predictions using uMelt parameters

## Recent Improvements

- **Hierarchical Clustering**: Group curves by similarity using agglomerative clustering with configurable k parameter. Clusters are visualized with distinct colors and labeled in the legend.
- **Bug Fixes**: Fixed normalization failure handling and x-axis label generation
